# Smart Calculator For Student Grades

## Description
#### This is a reusable Python script that collects students' grading data (i.e. test score, assignment score, and exam score), validates the inputs, performs calculations, and evaluates student performance based on predefined conditions.

### The program demonstrates the use of:
- Object-Oriented Programming (OOP)
- Input validation
- Arithmetic operations
- Comparison operators
- Logical conditions

---

## Features

1. Accepts input for:
 - Test score
 - Assignment score
 - Exam score

2. Validates user input by:
 - Checking for empty inputs
 - Ensuring values are numeric
 - Removing unnecessary spaces
 
3. Performs arithmetic Operation to;
 - Calculate total score
 - Calculates average score
 
4. Uses comparison operations to determine whether the student:
 - Passed
 - Failed
 
5. Uses Logical Conditions to check if the student qualifies for an award

6. Displays the result in a clean and readable format.

---

## How it works:

1. The program prompts the user to enter:
 - Test score
 - Assignment score
 - Exam score
 
2. Each input is validated before being accepted

3. The program calculates:
 - Total score
 - Average score

4. The student's performance is evaluated based on:
 - Pass or fail status
 - Award qualification

5. The final result is displayed in a readable format

---

---

## How to use:
1. Ensure python 3 is installed on your system

2. Run the script:

```bash
python deliverable_2.py
```
3. Follow the prompts to input your scores

4. Hit enter to view results
